# Luna Eye - Landing Page

The landing page for Luna Eye - Kiskstarter project.

The project was created using:

- HTML5
- CSS3 (SCSS)
- JavaScript
- Figma
- BEM-methodology

Landing page has the responsive layout for:

- Large screens 2560px
- Full HD 1920px
- The design 1600px
- Notebook 1280px
- Tablet 1024
- Mobile (> 320px)
